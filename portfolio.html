My Work at Reelgood

I was at Reelgood for one year. I was one of five full time developers, and I worked predominantly on the front end. I worked on many major features:

Reelgood - About

This is the Reelgood about page, where you see the value proposition of Reelgood displayed beautifully. The version you see is the version I created.

Reelgood - All Content

This is the Reelgood all content page, which I created the first version of. It has since been reworked.

Reelgood - Sources

This is the Reelgood sources page, where you choose your sources. I wrote the first version, but it's since been reworked.

Reelgood - Account (must be logged in to view)

This is the Reelgood account page. This was the first page that I created for Reelgood.

My Work at Leanpub

I was at Leanpub for 8 months. I worked mostly on private projects. The public-facing areas of Leanpub that I worked on were Leanpub for Publishers, and various areas of Leanpub's book publishing workflow.

I wrote a lot of Ruby on Rails and React code for this job. I was writing components of large web apps which included handling data, creating user interfaces, and styling them. The photo above showcases the feature to add an editor, which I created. You used this feature to manage editors on a Leanpub Publisher account.
