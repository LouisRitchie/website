Concatenating 2 or more .mp4s end-to-end with ffmpeg, a linux audio/visual encoder

Alright, so I just got finished concatenating some .mp4 files using ffmpeg. It took me some time to get it working properly, so I'm going to record my setup here for future reference.

1. transform each .mp4 into an intermediate .ts file 

ffmpeg -i myfile1.mp4 -c copy -bsf:v h264_mp4toannexb -f mpegts intermediate1.ts

2. concatenate all of the intermediate files 

ffmpeg -i "concat:intermediate1.ts|intermediate2.ts" -c copy -bsf:a aac_adtstoasc output.mp4
